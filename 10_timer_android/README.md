# Preset Timer Android

Simple Android timer app for saving named countdown presets.

## Features

- Save timer presets with name, minutes, and seconds
- Edit or delete saved presets
- Start, pause, resume, and reset the selected timer
- Sound and vibration when the timer finishes

## Build

```powershell
.\gradlew.bat assembleDebug
```

The debug APK is generated under `app/build/outputs/apk/debug/`.
